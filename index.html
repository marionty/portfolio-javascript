About me

Bonjour et bienvenue dans mon univers !

Je suis un véritable caméléon professionnel, ayant passé environ 10 ans dans le monde chatoyant du prêt-à-porter et même quelques escapades dans le secteur médical. Mais un jour, j'ai décidé de prendre une tangente audacieuse vers le développement web et mobile.

Tout a commencé en mai 2022, lorsque j'ai eu la chance de participer à une initiation en HTML et CSS. C'était comme ouvrir un coffre au trésor rempli de balises énigmatiques et de styles éblouissants. J'ai été immédiatement captivée et j'ai su que je tenais là ma prochaine grande aventure.

Depuis février 2023, je suis en formation de développeur web et mobile. Je suis encore aux premiers pas de mon parcours, découvrant avec émerveillement les langages du front-end et du back-end. C'est comme explorer une jungle mystérieuse remplie de JavaScript, PHP et autres créatures de code étranges. Mais je suis prête à relever tous les défis !

En août, je commence un stage de deux mois et demi, une véritable expédition au cœur du monde professionnel du développement web. C'est l'occasion idéale pour moi d'explorer davantage ce domaine, de développer mes compétences et de vivre de nouvelles aventures. Je vous invite à me suivre dans cette incroyable aventure. Sur mon portfolio , vous découvrirez mes premières créations, fruits de mes explorations et de mon apprentissage. C'est un peu comme un carnet de voyage numérique où je partage mes découvertes avec vous.

Alors, attachez vos ceintures et préparez-vous à explorer le monde du web à mes côtés. Même si je suis une débutante, je suis animée d'une curiosité sans limites et d'une envie constante d'apprendre et de grandir. Ensemble, nous allons plonger dans cet océan numérique et découvrir de nouvelles contrées fascinantes !

Que l'aventure commence !

Skills

HTML

HTML(HyperText Markup Language) est le langage de balisage standard utilisé pour créer la structure et le contenu d'une page web. Il est utilisé pour définir la structure des éléments tels que les titres, les paragraphes, les liens, les images, etc.

CSS

CSS(Cascading Style Sheets) est un langage utilisé pour définir l'apparence et la mise en page d'une page web. Il permet de spécifier les couleurs, les polices, les marges, les bordures et d'autres propriétés visuelles des éléments HTML.

JAVASCRIPT

JavaScript est un langage de programmation côté client utilisé pour rendre les pages web interactives. Il permet d'ajouter des fonctionnalités dynamiques aux sites web, de manipuler le contenu de la page, de répondre aux actions de l'utilisateur et de communiquer avec le serveur.

PHP

PHP est un langage de programmation côté serveur principalement utilisé pour développer des applications web dynamiques. Il est intégré dans le code HTML et peut être utilisé pour générer des pages web dynamiques, interagir avec des bases de données et traiter des formulaires.

REACT

React est une bibliothèque JavaScript développée par Facebook. Elle est utilisée pour construire des interfaces utilisateur interactives et réactives. React utilise un concept appelé "composants" pour organiser le code et faciliter la réutilisabilité. Il est souvent utilisé pour créer des applications web à une seule page (Single-Page Applications).

SYMFONY

Symfony est un framework PHP open-source et robuste utilisé pour développer des applications web. Il suit le principe de conception du modèle MVC (Modèle-Vue-Contrôleur) pour séparer la logique métier, la présentation et la gestion des requêtes. Symfony offre une large gamme de fonctionnalités et de composants pour accélérer le processus de développement.

NODE JS

Node.js est un environnement d'exécution côté serveur basé sur le moteur JavaScript V8 de Chrome. Il permet d'exécuter du code JavaScript côté serveur, ce qui ouvre de nouvelles possibilités pour le développement web. Node.js est souvent utilisé pour créer des serveurs web rapides et évolutifs, ainsi que des applications réseau en temps réel.

WORDPRESS

WordPress est un système de gestion de contenu (CMS) open-source largement utilisé pour créer des sites web et des blogs. Il offre une interface conviviale pour la création et la gestion de contenu, ainsi qu'une multitude de thèmes et de plugins pour personnaliser et étendre les fonctionnalités du site.
